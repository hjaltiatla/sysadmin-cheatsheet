# Sysadmin Cheatsheet

A single-page reference covering the core knowledge needed for day-to-day sysadmin work.

**Live site:** [cheatsheet.hjalti.me](https://cheatsheet.hjalti.me)

## What's inside

* Networking fundamentals - OSI model, DNS, DHCP, protocols
* CIDR & subnetting
* Ports & VPN technologies
* Security - hardening, encryption, common attacks
* OS & commands - Linux, Windows/PowerShell, Active Directory
* Linux & Windows filesystem hierarchies
* Logs & monitoring
* Certificates & firewall
* DevOps tools - Docker, Kubernetes, Git, CI/CD
* Network deep dive - Wireshark, routing, VLANs
* Cloud - AWS and Azure core services
* M365 & Entra ID - Graph PowerShell, Exchange Online, Intune, device join
* Resources & tools - communities, utilities, diagnostics

## Usage

Open `index.html` directly in a browser or visit the live site. Use the sidebar to navigate sections or press `/` to search.

## Deployment

Hosted on [Cloudflare Pages](https://pages.cloudflare.com), connected to this repo. Pushing to `main` triggers an automatic redeploy.
